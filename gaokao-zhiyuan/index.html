Problem Statement

志愿填报，是一场信息严重不对称的高代价博弈

每年 6 月，1,000 万家庭要在 2-3 周内完成一个影响一生的决定。市面上的解决方案要么贵到离谱，要么数据不准；张雪峰之后，连最贵的那条路也失去了核心 IP。

¥

付费门槛过高

张雪峰套餐 ¥9,999–17,999，AI 工具 ¥300–600 起步，把 70% 的家庭直接挡在门外。

⚠

数据与体验割裂

夸克概率不准，掌上高考 UI 老旧，优志愿被 upsell 反噬。没有一个工具能把测评 → 推荐 → 深聊串成完整体验。

★

张雪峰之后的真空

2,800 万抖音粉丝、市场最强信任 IP 离场。家长群体的核心需求"有人告诉我该怎么选"无人承接。

⇋

家长与孩子的分歧

"你想学游戏设计，你妈让你学金融"——一个家庭两个决策者，没有任何工具帮他们达成共识。

User Personas

为谁而做：考生、家长、学长

这是一个供需两端都需要服务的双边市场。需求端是焦虑的考生与家庭，供给端是名校在读学长——他们既是顾问，也是品牌信任的来源。

User Journey & Pain Points

从出分焦虑到下志愿表的 4 步

出分日是漏斗的最大入口。免费工具把陌生流量转化为注册用户，AI 对话沉淀深度需求，最后由学长 1v1 完成转化。每一步都对应一个具体的痛点。

1

出分焦虑

查到分数的那一瞬间，1,000 万家庭同时打开手机，不知道这个分数能上什么。

痛点：3 秒决定打开哪个 App
2

冲稳保推荐

输入省份、分数、选科，立刻拿到三档院校列表（基于真实录取数据）。

痛点：竞品概率不准、UI 慢
3

AI 深度对话

有具体院校后，问"清华计算机到底怎么样"——AI 给出专业、就业、城市的多维分析。

痛点：通用 AI 没有行业数据
4

学长 1v1 咨询

问到深处转向学长预约：30 分钟微信语音，直接听过来人怎么说。

痛点：信任 = 真实经历
User Stories

用户到底要完成什么

从需求侧的考生与家长，到供给侧的学长，每个角色都有具体要被满足的任务。

作为 高三考生
我想 输入分数立刻看到冲/稳/保三档院校列表
这样 我能在出分当天就开始规划，不再在 30+ 个 App 之间反复横跳。
作为 焦虑家长
我想 用百元预算约一次清华学长的 1v1 咨询
这样 我不必花 1 万元找张雪峰团队，也能听到名校在读生的真实建议。
作为 高三考生与家长
我们想 各自独立做一份测评，看到双方的差异在哪
这样 我们能在饭桌上少吵一个月，把分歧变成可以讨论的具体选项。
作为 迷茫的考生
我想 用左右滑卡片快速比较专业，最后得到一份"志愿人格"
这样 选专业不再是 30 道严肃问卷，而是 5 分钟就能完成的轻量游戏，还能发朋友圈。
作为 清华在读学长
我想 把空闲时间变成每单 ¥70–140 的真实收入
这样 我不用备课、不用应付陌生人，就靠分享自己的真实专业经历挣零花钱。
Features & Demo

免费工具承接流量，付费学长承接转化

产品矩阵分三层：获客（AI 对话 / 冲稳保 / 测评）、留存（完整报告 / 各省对比）、变现（学长咨询 / 全程辅导包）。每一层都有清晰的引流到下一层的钩子。

AI
💬

AI 志愿对话

基于 Claude Haiku + 录取数据 RAG，覆盖专业介绍、分数查询、就业前景、选科建议。注册前 20 轮 / 注册后 50 轮。

转化钩子： "想听清华学长亲口说？点这里"
CORE
⚡

冲/稳/保推荐

输入省份+分数+选科，3 秒生成三档院校列表。基于 18,000+ 条 2024 湖北省专业级录取数据，扩展中。

目标 SLA： 出分日 3 秒出结果
UX
❤

专业 PK 测评

Tinder 式左右滑卡片 + 场景题 + 硬性条件，5 分钟生成"志愿人格"标签和匹配专业。带分享图，K 因子 0.6–1.2。

设计原则： 滑动让人上瘾不退出
UX
♂♀

家长 vs 孩子对比

家长和孩子各自独立测评，可视化展示双方差异。一次场景天然触达两个用户，K 因子 1.5–2.0。

传播力： 双倍传播 + 家长群天然分发

性能 / Performance

出分日预期 1,000 万级峰值流量。冲稳保查询走预计算缓存，目标 P95 < 3 秒。AI 对话用 Haiku 模型，单次成本 < ¥0.01。

可扩展性 / Scalability

数据层按省份分表，先湖北 18K 条跑通，5 月底扩展到全国主要省份。AI 调用按用户限额，每天 20–50 轮，分享解锁。

护城河 / Moat

真正的壁垒不是数据（可被复制），而是名校学长网络。Phase 1 清华 30 人 → Phase 5 全 985+热门 211 共 230+ 人。
